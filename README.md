# Research Group Website

This is the source code for the Research Group website, hosted on GitHub Pages.

## Project Structure

- `index.html`: Home page with recent research progress.
- `members.html`: Team members introduction.
- `publications.html`: List of publications and work.
- `news.html`: Archive of news items.
- `assets/css/style.css`: Main stylesheet.
- `assets/images/`: Directory for images.

## Development

This is a static website built with vanilla HTML and CSS. No build process is required. To view the site locally, simply open the `.html` files in your web browser.

## Deployment

Push changes to the `main` (or `master`) branch to deploy to GitHub Pages.
